# 使用说明
# Tutorial

------------------------------------------------------------------------------

>问：如何使用这个压缩包？
>Q: How to use this zip file?

>答：将该包解压，把“Ender Ring Data Pack”放入存档文件夹中的 datapacks 中，将 resources.zip 直接放入存档文件夹中。
>A: Unzip it, put "Ender Ring Data Pack" in the "datapacks" folder in your save folder, and put resources.zip directly into your save folder.

----------------------------------------------------------------------------------

>问：在游戏里如何操作？
>Q: How to play it in the game?

>答：首先前往末地，将一个海洋之心和一个末影箱丢到一处，即可获得新物品末影环。将末影环拿在手中时会有悬浮文字提示当前模式（当前要放置的方块），潜行并右击可切换模式，右击地面即可在上方放置对应的暂时性方块。放在左右手各自有不同模式。离开暂时性方块一定距离后，它会自动消失。
>A: To start with, you need to reach the End dimension and throw a Heart of the Sea and an Ender Chest at the same place to get a new item: Ender Ring. With Ender Ring in your hand, there will be tooltips displaying the current mode of it (the current block that can be placed). Shift and right-click to switch the mode, directly right-click the ground to place the corresponding temporary block on it. When in mainhand and offhand, there are different variants of modes. The temporary block will disappear as soon as there is no player in its range.
